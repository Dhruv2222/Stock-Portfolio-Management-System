-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Running the JAR file:

1. Run the command java -jar (filepath of Assignment6.jar) while in the res folder.
2. Make sure that the data folder which contains the csv file is in the res folder (Both data folder and jar file should be on the same level).

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

External Library:

1. We have used the JFreeChart library for plotting the bar graphs and line graphs.
Link: https://jfree.org/jfreechart/

The jar files of the JFreeChart library need to be in the lib folder.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

1. To run the Graphical User Interface:
--> Run the command java -jar (filepath of Assignment6.jar) while in the res folder.

2. To run the Text-Based interface:
--> Run the command java -jar (filepath of Assignment6.jar) and add "textbased" at the end of the path.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

--> DATE input specification:

1. For the text-based view, date should be entered in the specified format (YYYY-MM-DD).

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

--> List of Stocks the program supports:

1. All stock tickers supported by API: Alpha Vantage.
2. Refer listOfStockTickers.csv file in the data folder within the res folder.
